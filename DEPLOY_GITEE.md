# Gitee Pages 部署指南

## 为什么选择 Gitee Pages？

✅ 国内访问速度极快
✅ 完全免费
✅ 自动部署
✅ 支持自定义域名
✅ 与 GitHub 类似的使用体验

## 部署步骤

### 1. 注册并登录 Gitee

访问：https://gitee.com
使用 GitHub 账号或邮箱注册

### 2. 导入 GitHub 仓库

1. 登录后点击右上角 **"+"** → **"从 GitHub / GitLab 导入仓库"**
2. 选择您的 GitHub 仓库：`HEKK777/my-react-app`
3. 点击导入，等待完成

### 3. 启用 Gitee Pages

1. 进入导入的仓库页面
2. 点击顶部菜单的 **"服务"** → **"Gitee Pages"**
3. 点击 **"启动"** 按钮
4. 选择部署分支：`main`
5. 选择部署目录：留空（根目录）
6. 点击 **"启动"** 等待部署

### 4. 获取公网地址

部署成功后，您会获得类似这样的地址：
```
https://hekk777.gitee.io/my-react-app
```

这个地址在国内可以流畅访问！

## 更新网站

每次您推送代码到 GitHub 后：

### 方法一：手动更新（Gitee 免费版）
1. 在 Gitee 仓库页面点击 **"服务"** → **"Gitee Pages"**
2. 点击 **"更新"** 按钮
3. 等待几秒钟即可

### 方法二：自动同步（Gitee 专业版）
- Gitee 专业版支持自动同步 GitHub 仓库
- 费用：约 60 元/年

## 注意事项

⚠️ Gitee Pages 要求：
- 仓库必须公开
- 不能包含大型二进制文件
- 单个仓库大小限制 1GB

✅ 本项目完全符合要求！
